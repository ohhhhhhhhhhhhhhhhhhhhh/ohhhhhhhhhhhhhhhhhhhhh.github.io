从0开始构建你的个人博客(Hexo)

简介

Hexo官网地址

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

安装

安装过程非常简单，只需要在您的电脑上下载如下软件即可：

  • Node.js (Node.js 版本需不低于 10.13，建议使用 Node.js 12.0 及以上版本)
  • Git
  • Hexo

部署到Github

Step1. 设置Git用户名和邮箱(第一次使用Git的话)

这里的用户名和邮箱随便填，但注意要有冒号。

Step2： 生成本地公钥

打开文件夹“C:\Users\用户\ .ssh ， 并复制id_rsa.pub中的内容(可用记事本打开)。

Step3: 在Github上创建SSH密匙

在GitHub设置(Settings)界面，进入SSH and GPG keys，点击 New SSH key。

Title 随便，粘贴复制的 id_rsa.pub 内容到 Key 中，点击 Add SSH key 完成添加。

Step4: 创建Github仓库

GitHub 主页右上角加号 -> New repository：

  • Repository name 中输入 用户名.github.io
  • 勾选 “Initialize this repository with a README”
  • Description 选填

填好后点击 Create repository 创建。

创建后默认自动启用 HTTPS，博客地址为：https://用户名.github.io
