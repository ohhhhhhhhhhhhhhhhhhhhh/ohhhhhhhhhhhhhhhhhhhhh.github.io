树状数组&线段树

树状数组

树状数组可解决**1.单点修改，单点查询，2.区间修改，单点查询，3.区间查询，区间修改（复杂捏）**等线段树可解决的大部分问题，二者时间复杂度相近（都是O（nlogn）），线段树功能性强大但码起来耗时间且复杂，树状数组拓展性低但使用方便。

问题引入

如题，已知一个数列，你需要进行下面两种操作：

  • 将某一个数加上 x
  • 求出某区间每一个数的和

来源：P3374 【模板】树状数组 1 - 洛谷

一种暴力的想法是O（1）时间复杂度进行单点修改，然后O（n）时间复杂度更新该数组的前缀和，但如果重复进行m次查询，总算法时间复杂度为O（mn），显然是会TLE的。于是，就有了接下来树状数组（线段树）的O（mlogn）做法。

lowbit(x)运算

在进行树状数组的介绍前，我们需要知道什么是lowbit（x），因为树状数组的各项操作都是建立在这基础之上滴。

所谓lowbit(x)，就是求一个非负整数x在二进制表示下，最低位的1和它后面的0所构成的数字，例如：

lowbit(44)=lowbit((101100)2)=(100)2=4lowbit(44) = lowbit((101100)_2) = (100)_2 = 4 lowbit(44)=lowbit((101100)2​)=(100)2​=4

好的，那么我们怎么求这lowbit(x)呢，其实很简单，一条式子足够了。

lowbit(x)=x&(−x)lowbit(x) = x \& (-x) lowbit(x)=x&(−x)

解释：

44的二进制表示为(101100)2(101100)_2(101100)2​，-44的二进制表示为(010100)2(010100)_2(010100)2​，然后我们把44和-44进行按位与运算得到二进制数(000100)2(000100)_2(000100)2​，也就是十进制的4。

注意： -44的二进制表示是44取反后+1。

树状数组结构分析

下面是一个树状数组的结构，至于其为何长成这样，先不着急了解。

对于一个长度为nnn的数组a[]a[]a[],我们引入一个数组t[x]t[x]t[x]表示以xxx为根的子树中叶子节点的值之和。

t[1] = a[1]

t[2] = t[1] + a[2]

t[3] = a[3]

t[4]=t[2]+t[3]+a[4]=t[1]+a[2]+t[3]+a[4]=a[1]+a[2]+a[3]+a[4]

…

单独这样来看，似乎找不到什么规律，但是，如果结合前边提到过的lowbit(x)运算，我们可以发现整个树就是通过它生成的，其基本性质如下：

  • 树状数组中节点t[x]t[x]t[x]的父亲节点为:t[x+lowbit(x)]t[x+lowbit(x)]t[x+lowbit(x)]。

  • 相邻子树的根节点对应下标的差值也为lowbit，如:

    • t[6]=t[7−lowbit(7)]t[6] = t[7-lowbit(7)]t[6]=t[7−lowbit(7)]
    • t[4]=t[6−lowbit(6)]t[4] = t[6-lowbit(6)]t[4]=t[6−lowbit(6)]
    • …

  • 求前n个元素的和：

    sum=t[n]+t[n−lowbit(n)]+t[(n−lowbit(n))−lowbit(n−lowbit(n))]sum = t[n] + t[n-lowbit(n)] + t[(n-lowbit(n)) - lowbit(n-lowbit(n))] sum=t[n]+t[n−lowbit(n)]+t[(n−lowbit(n))−lowbit(n−lowbit(n))]

  • 求区间[l,r][l,r][l,r]之间元素的和，就做个差分

    sum(l,r)=sum(r)−sum(l−1)sum(l,r) = sum(r) - sum(l-1) sum(l,r)=sum(r)−sum(l−1)

利用以上几点性质，关于树状数组的一写简单操作我们就可以理解啦。

P3368 【模板】树状数组 2

如题，已知一个数列，你需要进行下面两种操作：

  1. 将某区间每一个数加上 x*；
  2. 求出某一个数的值。

输入格式

第一行包含两个整数 N、M，分别表示该数列数字的个数和操作的总个数。

第二行包含 N 个用空格分隔的整数，其中第 i个数字表示数列第 i项的初始值。

接下来 M 行每行包含 2 或 4个整数，表示一个操作，具体如下：

操作 1： 格式：1 x y k 含义：将区间 [x,y] 内每个数加上k；

操作 2： 格式：2 x 含义：输出第 xx 个数的值。

输出格式

输出包含若干行整数，即为所有操作 22 的结果。

解析

关于区间修改，我们需要引入一个差分的概念。并使用树状数组维护a[]的差分数组t[]，所谓差分数组，就是每一项都为原数组前后两项之差，其具有如下性质：

  • t[i]=a[i]−a[i−1]t[i] = a[i] - a[i-1]t[i]=a[i]−a[i−1]
  • a[k]=∑i=1kt[i]a[k]=\sum\limits_{i=1}^{k} t[i]a[k]=i=1∑k​t[i]

于是乎，在区间[L,R][L,R][L,R]上加kkk时，我们只需要令t[L] += k, t[R+1] -= k,这样操作可以保证:

∑i=1R+1t[i]≡a[R+1],∑i=1jt[i]=a[j]+k,(L≤j≤R).\sum\limits_{i = 1}^{R+1}t[i] \equiv a[R+1], \\ \sum\limits_{i = 1}^{j}t[i] = a[j]+k, (L \leq j \leq R). i=1∑R+1​t[i]≡a[R+1],i=1∑j​t[i]=a[j]+k,(L≤j≤R).
